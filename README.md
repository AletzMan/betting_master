Betting Master es una aplicación construida con Next.js que te permite disfrutar de apuestas deportivas en línea y hacer un seguimiento de tus pronósticos y resultados.  Mantente actualizado con los últimos resultados de los partidos. Accede a estadísticas detalladas de equipos y jugadores para tomar decisiones informadas al realizar tus apuestas.

## Tecnologías Utilizadas

Betting Master se ha construido utilizando las siguientes tecnologías:

- Frontend: [Next.js](https://nextjs.org/), [React](https://es.reactjs.org/), [TypeScript](https://www.typescriptlang.org/), [Zustand](https://zustand-demo.pmnd.rs/), [SASS](https://sass-lang.com/)
- Backend: [Next.js](https://nextjs.org/), [Axios](https://axios-http.com/)
- Base de Datos: [Firebase Storage](https://firebase.google.com/docs/storage)
- Autenticación: [Firebase Authentication](https://firebase.google.com/docs/auth)

## Instalación
Para instalar Betting Master en tu entorno local, sigue estos pasos:

1. Clona el repositorio:

```bash
git clone https://github.com/AletzMan/betting_master
```

2. Entra en el directorio del proyecto:

```bash
cd betting_master
```

3. Instala las dependencias del proyecto:

```bash
npm install
```
4. Configura las variables de entorno:

    Crea un archivo .env en el directorio raiz y configura tus variables de entorno, incluyendo las claves de acceso de Firebase, siguiendo como ejemplo el archivo example.emv que se encuentra en el repositorio.

5. Inicia el servidor:

```bash
npm run dev
```

6. Abre tu navegador y visita http://localhost:3000 para acceder a Betting Master.

## Uso

Betting Master te permite:

- Iniciar sesión con tu cuenta de Google, Facebook o X.
- Realizar apuestas con familiares y amigos de los partidos de futbol.
- Hacer un seguimiento de tus pronósticos y resultados.
- Ver estadísticas de equipos y partidos, asi como resultados en vivo de los encuentros.

¡Diviértete con Betting Master!
